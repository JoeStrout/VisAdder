# VisAdder
Visualization/simulation app for tiny transformers that can do decimal addition

## Background

I'm interested in deeply understanding how transformers (and potentially other LLM architectures) work.

But it makes sense to start small.  What's the tiniest transformer that can show interesting behavior?  I think the answer is probably these 10-digit adders from the [AdderBoard contest](https://github.com/anadim/AdderBoard).

So, I will be starting with the **Litzenberger Adder**, a tiny transformer (available [here](https://gist.github.com/alexlitz/0d5efbccf443fb0e8136b8f5bd85140a)) that perfectly adds 10-digit numbers with only 36 free parameters.
